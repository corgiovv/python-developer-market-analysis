# Python Developer Market Analysis

## Overview

This project presents an exploratory data analysis of the Stack Overflow Developer Survey 2025 dataset.

The dataset contains 49,191 developer responses from around the world.  
The goal of this analysis is to explore Python adoption, developer experience, salary distribution, education background, and high-income remote workforce segmentation.

---

## Dataset

Source: Stack Overflow Developer Survey 2025  
Total respondents: 49,191  

The dataset includes information about:
- Programming languages used
- Professional experience
- Remote work status
- Education level
- Compensation
- Country of residence

---

## Dataset Access

The original Stack Overflow Developer Survey dataset is not included in this repository due to file size limitations.

To run the notebook locally, download the dataset from the official source and place the following files into the `data` directory:

- survey_results_public.csv
- survey_results_schema.csv

The project structure expects the files to be located at:

data/survey_results_public.csv
data/survey_results_schema.csv
